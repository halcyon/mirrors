Test­ing the setup



Once these source files are in place, we should be able to use jsonic as a lan­guage in DrRacket. For instance, if we run this source file:

jsonic-test.rkt

#lang jsonic
// a line comment
[
  @$ 'null $@,
  @$ (* 6 7) $@,
  @$ (= 2 (+ 1 1)) $@,
  @$ (list "array" "of" "strings") $@,
  @$ (hash 'key-1 'null
