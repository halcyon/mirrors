Domain-specific languages



As pro­gram­mers, the idea of mak­ing our own domain-spe­cific lan­guages (aka DSLs) may be new. But we’ve been using them for a long time.



What qual­i­fies as a DSL? A nota­tional sys­tem for cer­tain data, or oper­a­tions on data, that’s not intended to be used as a gen­eral-pur­pose lan­guage.



We could quib­ble about def­i­n­i­tional details. But let’s not. The point of this list is to help those who are inter­ested in bring­ing DSLs into their projects or teams make the case that even if the ter­mi­nol­ogy is new, DSLs are estab­lished in pro­gram­ming prac­tice.



Com­mon DSLs



When first pre­sented with the idea of a DSL, a con­trar­ian col­league will likely inter­ject: “Use a DSL? We’ve never done that before.”



The response: “Of course we have. For instance—”



Coq lan­guage for prov­ing math­e­mat­i­cal the­o­rems



CSS for­mat for stor­ing & relat­ing visual-styling instruc­tions for web pages



HDL a cat­e­gory of lan­guages for mod­el­ing hard­ware com­po­nents



HTML markup lan­guage for web doc­u­ments



Julia lan­guage for sci­en­tific & tech­ni­cal com­put­ing



MAT­LAB lan­guage for sci­en­tific & tech­ni­cal com­put­ing



awk text-pro­cess­ing util­ity on Unix-like oper­at­ing sys­tems



bash shell pro­gram on Unix-like oper­at­ing sys­tems



lex/yacc tools for mak­ing tok­eniz­ers & parsers



make tool for con­fig­ur­ing soft­ware builds



printf embed­ded lan­guage for for­mat­ting strings



Post­Script page-descrip­tion lan­guage



R lan­guage for sta­tis­ti­cal com­put­ing



reg­u­lar expres­sions embed­ded lan­guage for match­ing strings



SQL lan­guage for data­base access and man­age­ment



TeX high-level page-descrip­tion lan­guage



XML markup lan­guage, typ­i­cally for doc­u­ments

Racket-imple­mented DSLs



Tem­porar­ily mol­li­fied, the con­trar­ian col­league will then say “OK, a DSL is not such a crazy idea. But why should we use Racket to make a DSL? Who’s ever done that?”



The response: “Many have. For instance—”



HtDP teach­ing lan­guages sup­port­ing the How to Design Pro­grams text­book



brag lan­guage for gen­er­at­ing parsers from BNF gram­mars



datalog lan­guage for logic pro­gram­ming



web-server lan­guage for mak­ing web appli­ca­tions



shill script­ing lan­guage with priv­i­lege con­trols



Pollen lan­guage for mak­ing dig­i­tal books (like this one)



Pyret lan­guage for teach­ing intro­duc­tory pro­gram­ming



Rosette lan­guage for cre­at­ing pro­gram ver­i­fi­ca­tion and syn­the­sis tools



Scrib­ble lan­guage for gen­er­at­ing Racket doc­u­men­ta­tion



Slideshow lan­guage for gen­er­at­ing inter­ac­tive slide pre­sen­ta­tions
